# 📊 Retail Sales Analytics Dashboard | Power BI

## 📌 Project Overview

This project is an interactive Retail Sales Analytics Dashboard built using Microsoft Power BI. The dashboard helps analyze sales performance across customers, products, categories, regions, and shipping modes.

---

## 🚀 Dashboard Features

- 📈 Total Sales KPI
- 📦 Total Orders KPI
- 👥 Total Customers KPI
- 💰 Average Sales per Order
- 💵 Average Sales per Customer
- 📊 Sales Trend Analysis
- 🛍️ Sales by Category
- 🌍 Sales by Region
- 🏆 Top 10 Products
- 👤 Top 10 Customers
- 🎯 Interactive Slicers

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- DAX
- Power Query
- Excel Dataset
