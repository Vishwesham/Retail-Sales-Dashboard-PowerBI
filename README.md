# Retail-Sales-Dashboard-PowerBI
Interactive Retail Sales Dashboard built using Microsoft Power BI.
## Overview

This project is an interactive Retail Sales Dashboard built using Microsoft Power BI.

## Features

- Total Sales KPI
- Total Orders KPI
- Total Customers KPI
- Average Sales per Order
- Average Sales per Customer
- Sales Trend by Year
- Sales by Category
- Sales by Region
- Top 10 Products
- Top 10 Customers
- Interactive Slicers

## Tools Used

- Microsoft Power BI
- DAX
- Data Modeling
